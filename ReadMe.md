# Arithmetic Expression Evaluator
## EECS 348 term project
This is the repository for our Software Engineering I term project.

### What it does:
This is a simple expression evaluator that supports + (addition), - (subtraction), * (multiplication), / (division), % (modulus), ** (exponentation), and unary + and -.

## How to use it:
Please see our user manual in the docs folder to get started!
